How to use CSS in HTML File
============================
1. inline CSS (style attribute) (Restricted to 1 Element) (not re-usable)
    <div style="color:blueviolet;font-size: 24px;">Inline CSS uses style attribute</div>

2. internal CSS (style Tag)  (Restricted to 1 HTML File)
   <style></style>

3. external CSS
   a. Write css code in a Separate .css file
   b. include that .css file in HTML file using <link>



How to use Bootstrap in HTML Files
===================================
-Bootstrap is a CSS Library
-PreDefined CSS Classes
-Maintained by Twitter
-Bootstrap 5 is the Latest version
-Official Bootstrap Link: 
 https://getbootstrap.com/docs/5.3/getting-started/introduction/

 Add CDN Link to <head></head> of HTML File
 <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.8/dist/css/bootstrap.min.css" rel="stylesheet">


Types Of Javascript
=====================
1. Internal Javascript 
   <script></script>  Page Level  
   
2. External Javascript (Application/Project Level)
   a. Create a separate .js file and write javascript code
   b. include that .js file in HTML file using <script>


How to write and Execute javascript Code
=========================================
1. Browser (FrontEnd Development)
   a. Create a separate .js file and write javascript code (addition.js)
   b. include that .js file in HTML file using <script> (addition.html)
        <script src="addition.js"></script>
   c. Open HTML File in browser ( alt + b )

2. NodeJS (BackEnd Development)
   a. Create a separate .js file and write javascript code (addition.js)
   b. run the javascript file with below command
      node <filename>.js

Note: Every Javascript code that runs on browser might not work in nodejs, and vice-versa


Javascript Objects/Modules
==========================
Both Browser & NodeJS : console, Math , Array , Objects, Number, JSON

Only Browser, Not NodeJS :   document, window 

Only NodeJS, not Browser : fs , os , http , url
