Window Object methods
=====================

confirm() : User Confirmation
            Message + Ok-true + Cancel-false

prompt() : To Collect User Input 
           Message + InputBox + Ok, Cancel


Note: window object methods can be directly used
      window.alert()

undefined vs null
==================
-when a variable is declared without any value, javascript assigns 'undefined' value to that variable
-null is an intensional value assigned by developers.
-typeof 'undefined' is undefined, typeof 'null' is object


Note: Primitives are compared by value, non-primitives are compared by address
